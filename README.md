# Alamasyarie - Racket Project

This is a Racket language project.

## Prerequisites

You need to have Racket installed on your system. You can download it from:
- [Racket Official Website](https://racket-lang.org/)

## Getting Started

To run the main program:

```bash
racket main.rkt
```

## Project Structure

- `main.rkt` - Main entry point of the application

## About Racket

Racket is a general-purpose programming language and a platform for programming language design and implementation. It's a descendant of Lisp and Scheme.
